# 📝 Task App (PWA)

Aplicación web para gestión de tareas desarrollada con HTML, CSS y JavaScript.

## 🚀 Demo
👉  https://berenicemartinez-dev.github.io/task/

## 📱 Características

- Agregar y eliminar tareas
- Marcar tareas como completadas
- Persistencia con LocalStorage
- Notificaciones visuales (toast)
- Interfaz limpia y responsive
- Instalable como PWA (Progressive Web App)

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Service Worker
- Manifest.json
